# Laboratorio 2 AREP

Este repositorio se hace con el proposito de cumplir con la solucion del problema planteado por el profesor, que consistia en realizar el calculo de la media y la desviación estandar, almacenando estos datos en una LinkedList, sin embargo esto se desplego en heroku, en una aplicacion web

## Diseño

### Introducción al problema

El enunciado nos plantea el reto de escribir un programa con el que se pueda calcular la media y la desviacion estandar de una cantidad n de numeros reales, sin embargo este nos exige desarrollarlo por medio de una estructura de datos llamada LinkedList, el cual nos explica su funcionamiento en la guia, adicionalmente este pide desarrollarlo con ayuda de Maven y GIT, adicionalmente este pide que se despliegue la aplicacion en Heroku por lo cual el link a la App esta adelante. 

### Desarrollo de la solución

Se presento la misma solución planteada que en el laboratorio 2 pero se realizo el respectivo despliegue en Heroku.
